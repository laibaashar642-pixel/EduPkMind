📌 Overview
This project is part of my journey as a 4th-semester IT student exploring Python and Django.
The goal is to build a functional web application while learning about backend development, Git workflow, and deployment.
⚙️ Features
- User authentication (login/signup)
- Database integration with Django ORM
- Dynamic routing using Django’s MTV architecture
- Responsive templates with HTML/CSS
- Git version control for project management
🛠️ Tech Stack
- Backend: Django (Python)
- Database: SQLite (default) / PostgreSQL (future deployment)
- Frontend: HTML, CSS, Bootstrap
- Version Control: Git & GitHub
🚧 Challenges Faced
- Understanding Django’s MTV pattern
- Handling migration errors and database setup
- Managing multiple apps and URL routing
- Learning Git commands for commits, branches, and pushes
- Preparing for deployment (Heroku/AWS/Docker exploration)
📖 Git Workflow (Cheat Sheet)
# Initialize repo
git init

# Add files
git add .

# Commit changes
git commit -m "Initial commit"

# Push to GitHub
git push origin main

# Create new branch
git checkout -b feature-branch
